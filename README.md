# ANTHONUS-bot

Bot Discord personnel développé pour l'apprentissage de Kotlin, Java et [JDA](https://jda.wiki), ainsi que pour automatiser quelques tâches du quotidien.

## Objectif du projet

Ce dépôt sert de terrain d'expérimentation autour de :
- Kotlin sur la JVM
- L'API Discord via JDA
- L'intégration de bibliothèques tierces (audio, HTTP, configuration, logs)

## Prérequis

- Java 25 ou supérieur

## Installation et démarrage

1. Télécharger [la dernière release](https://github.com/ANTHONUSS/ANTHONUS-bot/releases/latest).
2. Lancer le JAR :

```bash
java -jar ANTHONUS-bot-<version>.jar
```

## Développement

Compilation locale du projet :

```bash
./gradlew build
```
